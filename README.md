# Digitool

Welcome to **Digitool**! This is a modern, responsive, and dynamic web application built to provide a great user experience across all devices.

## 🚀 Overview

Digitool is designed with a focus on smooth interactions and dynamic content. A few core highlights of the project include:
- **Fully Responsive Layouts:** Carefully styled to look great on mobile, tablet, and desktop screens.
- **Dynamic Content:** Product cards and other data are loaded dynamically from external JSON sources, keeping the content fresh and easy to manage.
- **Smart Conditional Rendering:** The user interface reacts to your actions—like updating the cart count in the navigation menu, showing item badges, and smoothly handling "Buy now" buttons based on their state.

## 🛠️ Technologies Used

Under the hood, the project is powered by a solid stack of modern tools:
- **React (ES6)**
- **Tailwind CSS & DaisyUI** for styling and UI components
- **React Icons** for crisp vector graphics
- **React Toastify** for elegant notification popups
